# Eenvoudige Klantbeheer Applicatie

Ontwikkel een eenvoudige consoleapplicatie in C# die het beheer van klanten mogelijk maakt. De applicatie moet de volgende functionaliteiten bieden:

1.  **Klant Toevoegen:** Mogelijkheid om een nieuwe klant toe te voegen met een unieke ID, naam en e-mailadres.
2.  **Klant Weergeven:** Mogelijkheid om alle toegevoegde klanten weer te geven.
3.  **Klant Zoeken:** Mogelijkheid om een klant te zoeken op basis van zijn ID.

Gebruik properties om de klantgegevens (ID, Naam, E-mail) te beheren. Zorg ervoor dat de code netjes gestructureerd is en gebruikmaakt van duidelijke methoden.

## Opdracht

Implementeer de gevraagde functionaliteit.
